“这个词并不存在” Web Github

一位老哥在牛津英文词典上重新训练了GPT-2模型，现在我们可以随便瞎掰一个英文单词，由它给出编造的释义和例句，最恐怖的是，编造出来的解释还挺像样的。举例来说我输入hokuto(日文‘北斗’的罗马音)，它生成的释义是一种日本茶的名称，不懂的人真的可能被蒙进去。

Yolov5？

之前Yolo的作者说处于道德顾虑不更新Yolo了，大家一度觉得非常尴尬。最近一位老哥AlexeyAB出来更新了Yolov4，社区一度非常高兴，然而我这两天逛Github的时候，突然又发现了yolov5,研究了下这应该是这位老哥在自己的yolov3版本上做了些改动，融合进新的yolov4做出来的新模型。

虽然说大家这么踊跃是好事，但是如果承认这是v5，那整个迭代全乱套了，所以我还是倾向于这是一个魔改作品。真正的yolo系列由AlexeyAB维护，毕竟社区里一般认为AlexeyAB和原作者还是有些关系的。

在微型设备上运行神经网络 Reddit

之前看到reddit上老哥发了一个帖子，说他整了一个在微型设备上跑RNN的项目（大概16MHZ 的开发板） ,还是挺有意思的，目前太多神经网络的工作都越做越大越做越夸张，像新出的GPT-3论文，简直大到了令人发指的地步，偶尔有这种轻型的项目还是挺好玩的。
