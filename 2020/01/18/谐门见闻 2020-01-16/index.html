HuggingFace推出了新的高速分词包

我们的老朋友抱抱脸又给大家整活啦，这次是新的高速分词包tokenizers，底层用的是rust所以相当快，据称可以在20秒内完成对GB级文本的分词（在服务器CPU上)

然而目前似乎只支持英文，对我们没有太大卵用。

部署框架Cortex Github Reddit Medium

国外老哥们整了一个新的机器学习模型部署框架cortex,支持tf、pytorch、sklearn、xgb等模型，可以直接连通aws进行部署。

不过说实话，我并不喜欢一个部署框架把自己和某个云服务商强绑定，如果是AWS、谷歌云自己开发的部署工具，那我倒没意见，但是你一个第三方的部署框架，没必要做到这种程度的支持。考虑下工程方面的做法，也没有看见有同学直接命令行上操作一下就直接部署到云上的。我觉得这个还是应该解耦：服务构成归服务构成，部署用部署的工具。

就好像我起服务用的是tf-serving,tf-serving作为一个服务框架帮我把模型变成可调用的接口，但是之后我就可以想怎么做就怎么做，打成docker部署也好，放到集群上也好，具体部署方式

不过不论怎么说，cortex还是可以给我们之后做算法部署做一个参考。

self-attention机制与卷积之间的关联 Arxiv Github Reddit

ICLR 2020上的文章,阐述了self-attention机制与卷积之间的关联性，为何卷积能在一些情况下替代self-attention发挥作用。

比较好的是作者同时放出了论文的实现代码，有兴趣可以参观学习一波。

AutoGluon Github

来自亚马逊的机器学习自动化框架，基于mxnet实现，据称在网络结构、超参数搜索上有不俗的表现，我们要找个时间测试体验一下。
