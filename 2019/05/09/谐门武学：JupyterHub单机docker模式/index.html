挂载私人与公用的目录

jupyterhub-deploy-hub本身已经为你挂载了个人用户的目录，存在/home/jovyan/work下的文件都会被持久化保存。这些个人目录都是互相独立的，它们存在于/var/lib/docker/volumes/jupyterhub-user-{username}/_data这个文件夹下(假如docker的默认存储路径是/var/lib/docker的话）
